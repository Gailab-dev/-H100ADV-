2. 프로젝트 빌드
  eclipse 툴 기준
  1). 상단 메뉴 - project - clean
  2). (maven 최신 상태가 아니라면) 프로젝트 루트 폴터 - 마우스 오른쪽 버튼 클릭 - maven - update project
  3). 프로젝트 루트 폴더 - 마우스 오른쪽 버튼 클릭 - Run As - maven clean
  4). 프로젝트 루트 폴더 - 마우스 오른쪽 버튼 클릭 - Run As - maven install 
  5). 프로젝트 루트 폴더 - 마우스 오른쪽 버튼 클릭 - Refresh
  6). 프로젝트 루트 폴터 하위 폴더 중 target 폴터 하위 폴더에 *.war 파일 생성 확인
